# TinyAVG初始化

感谢您选择使用TinyAVG编写文字游戏。在正式开始之前，可能需要确认一些必须条件，以及做一些必要的准备工作。本指南将带领您完成这些工作。

## 第一小节：配置要求

TinyAVG只有在特定的环境下才可以正常运行。这一小节将帮助您检查您的计算机，确保它具备下表所示的一切必需条件。否则，TinyAVG将无法正常运行。

|条件分类|要求|
|-|-|
|操作系统|Windows 7 / Windows 8 / Windows 10（推荐）
|中央处理器（CPU）|市面上大多数商用CPU均可满足要求。|
|物理内存|至少128Mib|
|DirectX运行库|DirectX 11及以上|
